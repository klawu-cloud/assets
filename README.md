# Assets Repository

This repository stores static assets, primarily for email usage.

## Structure

- `email/branding/logos/` - brand logos used in email headers/footers
- `email/branding/icons/` - small icon assets for email UI blocks
- `email/templates/` - template-specific images (hero, banners, section visuals)
- `email/social/` - social/channel logos for email footers
- `email/misc/` - temporary or uncategorized email assets

## Naming

Use clear, stable file names, for example:

- `brand-logo-primary.png`
- `icon-support-24.png`
- `newsletter-hero-apr-2026.jpg`

## Recommended formats

- `PNG`: logos/icons with transparency
- `JPG`: photo-heavy visuals
- `SVG`: vector assets where email-client support is acceptable
